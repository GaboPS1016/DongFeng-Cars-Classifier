# DongFeng-Cars-Classifier 🚗⚡
Deep learning classification of 12 Dongfeng car models using a custom grayscale dataset and CNNs. Includes scraping, preprocessing, and training code.

## Models:
-Aeolus AX7
-Aeolus Haoji
-Aeolus Yixuan
-epi 007
-Fengguang 500
-Fengguang 580
-Fengguang S560
-Forthing Lingzhi M5
-Forthing T5
-Forthing T5 evo
-M-Hero 917
-Nammi Box
